CSS Transitions

Transition CSS Properties

  • transition-property: [css property name goes goes here... color, width, opacity, etc...];
  • transition-duration: [time of transition animation in seconds or milliseconds... 10s, 500ms, etc...];
  • transition-timing-function: [animation easing function... ease-in-out, ease-out, linear, etc...];
  • transition-delay: [time to delay before animation begins in seconds or milliseconds];

Transition Shorthand Property

  • transition: [transition-property], [transition-duration], [transition-timing-function], [transition-delay];
  • Example:
    transition: background-color, 1s, ease-in-out, 500ms;

Chaining Transitions in Steps

  • transition-property: background-color, width, left, border-radius;
  • transition-duration: 1s, 500ms, 0.2s, 250ms;
  • transition-timing-function: ease-in, ease-out, linear, ease-in-out;
  • transition-delay: 0.2s, 250ms, 0.1s, 100ms;

Shorthand: Chaining Transitions in Steps

  • transition: background-color 1s ease-in 0.2s,
    width 500ms ease-out 250ms,
    left 250ms ease-in-out 0.2s,
    border-radius 400ms ease-in 0.1s;
